## Usage
- Install NodeJS 8.0 or higher.
- Run `npm ci` in the root folder
- Run `npm start` in the root folder

Then open up your favorite browser and go to http://localhost:3001/

## Build Usage
- Install NodeJS 8.0 or higher.
- Run `npm ci` in the root folder
- Run `npm run build` in the root folder
